# 📖 JWPub Reader para Linux (No Oficial)

Una aplicación nativa para Linux diseñada para leer y estudiar archivos `.jwpub` con una interfaz limpia, moderna y altamente funcional. 

> ⚠️ **Nota de exención de responsabilidad / Disclaimer:** Esta aplicación **no** proporciona, aloja ni distribuye ningún archivo de publicación propiedad de JW.ORG. Es exclusivamente un lector de archivos locales `.jwpub` de uso personal y no oficial.

---

## ✨ Características Implementadas

La aplicación ha sido desarrollada enfocándose en el rendimiento y la comodidad durante el estudio personal. Actualmente el lector cuenta con:

*   **Compatibilidad Total:** Soporte para el **95%** de los archivos `.jwpub`.
*   **Estilo Visual Impecable:** Interfaz limpia, bonita y moderna con **sistema de temas** personalizable.
*   **Navegación Inteligente (Enlaces Recursivos):** Capacidad de abrir textos, referencias e índices de estudio directamente desde el panel de estudio de forma recursiva, ¡sin necesidad de salir de la publicación actual!
*   **Editor de Discurso Avanzado (Al 70%):** Herramienta integrada y funcional para la preparación de discursos, que ya incluye soporte completo para resaltados y estilos de texto.
*   **Estilos de Texto y Resaltado:** Soporte completo para estilos especiales (**negrita**, *itálica*, __subrayado__, ~~tachado~~) y un potente sistema de **resaltado**.
*   **Notas y Estudio:** Sistema completo de notas accesible desde el lector, el panel de estudio o mediante **ventanas flotantes**.
*   **Búsqueda Avanzada de 3 Niveles:**
    1.  **Palabras Exactas:** Encuentra la coincidencia literal de lo que buscas.
    2.  **Búsqueda General (Flexible):** Ignora tildes y diacríticos (por ejemplo, buscar *Jehova* encontrará tanto *Jehová* como *Jehova*).
    3.  **Orden Disperso:** Encuentra frases o combinaciones de palabras sin importar el orden en el que aparezcan en el texto, siempre que coincidan en el mismo fragmento.
